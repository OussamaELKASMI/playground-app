# Fluxboard

Fluxboard is a deliberately imperfect demo app for Cursor stress testing.

It looks like a tiny full-stack internal productivity tool:

- a task dashboard
- a simple auth flow
- a task API
- shared types
- a couple of tests

It is intentionally rough so AI coding sessions have something to do:

- naming is inconsistent
- error handling is uneven
- loading states are weak
- auth/session logic is suspicious
- tests are incomplete

Open this folder in Cursor and let the stress harness drive a fake work session against it.
